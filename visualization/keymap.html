This visualization is based on the ZMK firmware configuration for the Cygnus split keyboard with a 3x6 layout. Click on the layer buttons above to view different layers. You can also click directly on highlighted layer keys in the keyboard to switch layers.

The keyboard uses a 6-layer setup including Base layer (QWERTY with Swedish layout), Brackets, Quotes, Symbols, Arrows, and Numbers layers.
